Instrucciones de Uso

Consejos para buscar de forma rápida y precisa

1. Uso básico

Para comenzar, puedes escribir una palabra o frase en el campo Búsqueda libre. El sistema buscará coincidencias en el nombre del fallo, la carátula, el resumen y los sumarios.

2. Uso de filtros

Los filtros permiten acotar los resultados:

  • Año del fallo: escribe el año exacto (ejemplo: 2023).
  • Categoría: selecciona del menú desplegable.
  • Tribunal: empieza a escribir para ver sugerencias automáticas.
  • Subcategoría: similar al tribunal, admite sugerencias.
  • Palabras clave: añade términos relevantes para tu búsqueda.

3. Consejos para optimizar la búsqueda

  • Comienza con un término amplio y luego añade filtros para afinar resultados.
  • Utiliza palabras clave específicas para encontrar fallos concretos.
  • Si no encuentras resultados, elimina algunos filtros y vuelve a buscar.
  • En Búsqueda libre, evita artículos o preposiciones (por ejemplo, en lugar de "el despido", escribe solo despido).
  • Prueba diferentes sinónimos (ejemplo: indemnización y compensación).

4. Interpretación de resultados

Cada resultado muestra:

  • Nombre y carátula del fallo.
  • Tribunal y fecha.
  • Categorías, subcategorías y palabras clave.
  • Resumen y enlace para ver el documento completo.
  • Botón para ver sumarios detallados.
  • Enlace al documento completo para descarga.
